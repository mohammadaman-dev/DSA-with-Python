## 🧩 Problems Solved

### 1. Two Sum

- Platform: LeetCode
- Topic: Array
- Difficulty: Easy
- Approach: Brute Force
- Description:
  - Find two numbers in an array whose sum is equal to the given target.
  - Return the indices of those two numbers.

- Time Complexity: O(n²)
- Space Complexity: O(1)


### 2. Linear Search

- Platform: LeetCode
- Topic: Array
- Difficulty: Easy
- Approach: Linear Search
- Description:
  - Search for a target element in an array by checking each element one by one.
  - Return the index of the target element if found; otherwise, return -1.

- Time Complexity: O(n)
- Space Complexity: O(1)
